ABOUT

SaaS stands for “software as a service”. This refers to having an alternate way that a person can access software with the goal of provide storage.

SIMPLE DESCRIPTION

  • One of the most common forms
  • Some of the services that major cloud providers have are:
    • Application monitoring
    • Data analytics
    • Virtual Private Networks
  • Some Real World examples include:
    • Amazon Web Services
    • Google Cloud Platform

BENEFITS AND USES

  • Benefits
    • The processing power is supplied by cloud
    • The application can only be used once the user subscribes
    • You have to pay for what you use
    • It is easy to access the additional storage without having to install new hardware or software.
    • The updates happen automatically and are free of any charge.
    • The applications can be accessed by any internet device. That can be desktops, smartphones, or tablets.
    • The applications can be accessed from any location.
    • Finally, the applications can also be modified to suit anyone's needs.
  • Example of when it can be used in business:
    • Accounting and invoicing
    • Tracking sales
    • Planning
    • Performance Monitoring
    • Communications
